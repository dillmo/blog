Chrono Warriors Progress Update #1

November 12, 2015

The Chrono Warriors code is now officially underway! Funnynin and I have worked hard to prepare Chrono Warriors for collaboration, and it is finally ready. This first progress check is part of a weekly series that will be published each Thursday. Each will mention the latest improvements in the game’s codebase.

New this week

  • Got underway with PySFML, a simple multimedia library for use with Python
  • Scaffolding for player and movement logic
  • Created a basic map using Tiled, an open-source map editor. We are working hard to add logic for handling Tiled maps. Tiled provides us the flexibility to easily create new maps with custom tilesets, and is easy to use even for non-coders.
