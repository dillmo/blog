Chrono Warriors Progress Update #2

November 19, 2015

More work has been done on Chrono Warriors! This week’s focus has been rendering logic for maps. At this point, the game is completely capable of rendering any maps designers make. Just don’t start work yet. We need to flesh out a few specifics before maps are actually playable, rather than simply renderable.

New this week

  • Now renders Tiled maps. Tilesets should be placed in src/client/assets and maps should be placed in src/client/levels.
  • Now depends on pytmx, a library for parsing Tiled maps.
